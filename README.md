This project transforms raw data from OpenDOSM into actionable insights to identify health disparities across demographics and regions.




Key Features

Interactive Dashboard: Dynamic visualizations of death counts by state, gender, and ethnicity.



Statistical Rigor: Analysis of 7,854 records with mean imputation and right-skewed distribution modeling.



Advanced Visualizations: Heatmaps for intersectional analysis (Gender x Ethnicity) and trend charts for COVID-19 impact.



üí° Key Insights

COVID-19 Impact: A 28.8% spike in deaths was identified in 2021, marking a historical peak of 898,000 deaths.



Gender Disparity: Males consistently show higher mortality rates than females across all ethnic groups.



High-Risk Demographics: The Bumi_malay (Male) subgroup was identified as the highest mortality cluster with over 1 million deaths.
